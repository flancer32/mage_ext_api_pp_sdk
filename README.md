# mage_ext_api_pp_sdk
Magento extension ot use [PayPal PHP SDK](https://github.com/paypal/PayPal-PHP-SDK).

## Installation

Add to your `composer.json`:
    {
      "repositories": [
        {
          "type": "vcs",
          "url": "https://github.com/flancer32/mage_ext_api_pp_sdk"
        }
      ],
      "require": {
        "flancer32/mage_ext_api_pp_sdk": "dev-master"
      }
    }
